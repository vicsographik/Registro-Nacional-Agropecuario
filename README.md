## Diseño
- Basado en: Sistema de Componentes Infotec, adaptando la identidad visual del Sistema Nacional de Diseño.

## Desarrollo
- Metodología: BEM
- Estándares de Accesibilidad: Cumplimiento de pautas WCAG 2.1 Nivel AA.
- Unidades: Uso de rem para tipografía y espaciados, garantizando la escalabilidad del diseño.
- /css: Hojas de estilo organizadas por componentes.
- /assets: Recursos gráficos, scripts y logotipos institucionales.
